# 播客

# 翻转电台 Flip Radio

李厚辰

当代社会人们触哲学的机会并不罕见，却被贯穿在整个教育体制提倡的工具理性限制，我们要不陷入“了解这些有什么用”的困惑中而放弃，就因为面对无数人名、书名无从下手而感到气馁。下文将以个人角度将欧洲哲学史上最重要的几位思想家的观点结合起来，希望为不了解哲学的朋友们展示它的趣味、脉络。
—— 从欧洲思想家到游戏艺术：《翻转电台》导读

明确、尖锐、严肃、理想的知识分享电台，每周在微信群以语音的方式进行知识分享和讨论并整理剪辑成播客，指出了很多司空见惯、习以为常的观点和理论其中所潜藏的问题，给沉溺于社会规训、消费主义等建构事物中的当代人一种独特视角。

# 内核恐慌

Rio、吴涛

号称硬核，可也没什么干货。想听的人听，不想听的人就别听。

年更节目，确实如简介中所说硬核且没有干货。😂️

每隔几个月听两位主播聊聊各自最近的科技生活之于我，就如同茶馆之于老北京，即使不期待什么实质内容，关于科技的闲侃本身也是一种乐趣。

# Anyway.FM 设计杂谈

JJ Ying、Leon Gao

我们的目标是让你的听觉更懂视觉。

两位主播的主业是设计师，在我看来是内核恐慌的设计师版本，一档以设计师视角闲侃科技生活的播客。

# 声东击西

世界宽广，但人们被日常及社交媒体所限制。那就让作为记者的我们，帮你推开一些意想不到的窗户，看见不一样的世界，触及不一样的信息，激发不一样的思考。

这里讨论城市与未来，电影与现场，也谈论技术创新与人类生活的变革。

这里有一手的现场观察和体验，还有不定期出现的神秘嘉宾。

# 软件

# 1Password

iOS/Android/macOS/Windows/Browser Extension

密码保险箱，易用性和颜值在同类软件中都首屈一指，除了密码以外，还保管了我的软件许可、金融账户以及护照之类的私密文件。我选择的是 1Password 7 的 macOS 和 Windows 的买断版本（首发七折优惠）搭配 Dropbox 使用，而不是 Family Plan 订阅。

# BetterTouchTool

macOS

自动化工具。

# Castro

iOS

泛用型播客客户端，收集箱的设计很契合我的使用方式。

# Dropbox

iOS/Android/macOS/Windows/Web

# Eagle

macOS/Windows/Browser Extension

图片素材管理软件，配合浏览器插件极大提高搜集图片素材的效率，同时图片管理功能也非常完善，支持文件夹、标签、注释、原始链接、以及对图片信息的展示，同时还能从图片中提取出五种主色调。

# Eudic

iOS/Android/macOS/Windows

辞典，允许导入.mdx文件，功能强大。

# Reeder

# Scrivener

iOS/macOS/Windows

集素材整理、写作编辑、排版校对于一体的专业工具，适合作家、编辑等需要撰写长篇文本的文字工作者使用。

# Sketch

macOS

# Spark

iOS/Android/macOS

使用最舒适的泛用型邮箱客户端

# Surge

iOS/macOS

抓包工具，无需多言，就是好贵用。
Clash (Android/macOS/Windows)：开源代理软件，社区活跃，用户群不断扩大，很有潜力;
Charles (iOS/macOS)：抓包工具，功能完善;

# Things

iOS/macOS

任务管理应用，OmniFocus、Todoist、2Do 都有使用过，但最终还是选择了Things，最适合我目前的GTD使用习惯和流程，不打扰、不繁琐，界面简洁友好，对URL Scheme的支持也很到位，还能整合邮箱到GTD流程中来。

# Ulysses

iOS/macOS

Markdown 编辑器，支持轻度写作与博文编辑。
MWeb (iOS/macOS)：中国独立开发者 @oulvhai 的作品，用户体验仅次于 Ulysses，甚至在功能上比 Ulysses 更胜一筹;
Typora (macOS/Windows)：Markdown编辑器、阅读器，界面遵循极简主义风格，但支持的语法非常丰富。

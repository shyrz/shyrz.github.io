# 软件

  • 1Password (iOS/Android/macOS/Windows/Browser Extension)
    密码保险箱，易用性和颜值在同类软件中都首屈一指，除了密码以外，还保管了我的软件许可、金融账户以及护照之类的私密文件。我选择的是 1Password 7 的 macOS 和 Windows 的买断版本（首发七折优惠），而不是 Family Plan 订阅，搭配 Dropbox 使用。
  • BetterTouchTool (macOS)
    自动化工具。
  • Castro (iOS)
    泛用型播客客户端，收集箱的设计很契合我的使用方式。
  • Dropbox (iOS/Android/macOS/Windows/Web)
  • Eagle (macOS/Browser Extension)
    图片素材管理软件，配合浏览器插件极大提高搜集图片素材的效率，同时图片管理功能也非常完善，支持文件夹、标签、注释、原始链接、以及对图片信息的展示，同时还能从图片中提取出五种主色调。
  • Eudic (iOS/Android/macOS/Windows)
    辞典，允许导入.mdx文件，功能强大。
  • IINA (macOS)
    macOS 上最好用的视频播放器，没有之一。
  • iMazing (macOS)
  • iTerm 2 (macOS)
  • Keyboard Maestro (macOS)
    自动化工具。
  • LaunchBar (macOS)
    启动器，主力，完全替代了孱弱的 Spotlight。[1]
  • MarginNote (iOS/macOS)
  • Notion (iOS/Android/Web)
    新概念的笔记应用，前景和潜力都不错。
  • Office 365 (iOS/Android/macOS/Windows)
  • Reeder (iOS/macOS)
    最好用的RSS阅读器，可惜没有 Android 版本。
  • Scrivener (iOS/macOS/Windows)
    集素材整理、写作编辑、排版校对于一体的专业工具，适合作家、编辑等需要撰写长篇文本的文字工作者使用。
  • Sketch (macOS)
    设计必备。
  • Spark (iOS/Android/macOS)
    使用最舒适的泛用型邮箱客户端
  • Surge (iOS/macOS)
    抓包工具，无需多言，就是好贵用。[2]
  • Things (iOS/macOS)
    任务管理应用，OmniFocus、Todoist、2Do 都有使用过，但最终还是选择了Things，最适合我目前的GTD使用习惯和流程，不打扰、不繁琐，界面简洁友好，对URL Scheme的支持也很到位，还能整合邮箱到GTD流程中来。
  • Ulysses (iOS/macOS)
    Markdown 编辑器，支持轻度写作与博文编辑。[3]
